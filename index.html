🔒 Covering Your Tracks

✅

VPN Kill Switch

If the VPN drops, all download traffic is instantly blocked. Your real IP can never leak.

✅

WireGuard Encryption

All download traffic is encrypted. Your ISP sees data going to a VPN server — nothing more.

✅

Network Isolation

Only download clients use the VPN. Streaming and management apps connect normally — no speed penalty.

✅

Zero-Waste Hardlinks

Downloaded files are hardlinked, not copied. A 50 GB movie uses 50 GB total — not 100 GB.

🎬 Requesting a Movie or Show

▼

1. Open Jellyseerr and search for what you want.

2. Click Request. It adds to Radarr (movies) or Sonarr (TV).

3. All indexers are searched via Prowlarr. Results scored by custom formats.

4. Best release downloaded through VPN tunnel.

5. File hardlinked into library. Jellyfin picks it up. Bazarr fetches subtitles.

Total time: usually 5–30 minutes.

🎵 Adding Music

▼

Open Lidarr (:8686), search for an artist, add them. Uses MusicBrainz for metadata. Albums scored with Davo's custom formats: CD rips and preferred groups score highest.

📚 Books & Audiobooks

▼

Enable profiles in .env (COMPOSE_PROFILES=kavita,audiobookshelf), restart. Open LazyLibrarian (:5299), add authors. PostProcessor sorts ebooks to Kavita and audiobooks to Audiobookshelf every 10 minutes.

🎮 Games

▼

Open QuestArr (:5002), search, pick a release. A background script (game-import.sh) moves completed downloads to the library every 15 minutes with 9 safety checks.

🔄 How Upgrades Work

▼

Every 15–30 minutes, RSS feeds are checked. If a release scores higher than your current file, the upgrade downloads automatically. Old file replaced, new one hardlinked. Keeps improving until the quality ceiling is reached.

💡 Check VPN First

▼

If downloads aren't working, start with arr vpn. Shows tunnel status, public IP, and whether download clients are reachable.

💡 Run Doctor Regularly

▼

arr doctor self-heals: missing dirs, outdated binary, missing env vars, wrong permissions. Run after updates.

💡 VPN Group Restart

▼

arr restart vpn restarts Gluetun, waits 10s for tunnel, then restarts both download clients in the correct order.

💡 Back Up Before Updates

▼

arr backup before arr update. Timestamped tar.gz of all configs. arr backup --prune 5 keeps only the 5 most recent.

💡 Understanding Hardlinks

▼

A 50 GB movie has two filenames but exists once on disk. One in downloads (for seeding), one in the library (for streaming). When the torrent is removed, the library copy persists. Works because all services mount the same root directory.

Jellyfin

Kill Switch Rules

Default policy: DROP. Only three types of outbound traffic allowed:

  • VPN tunnel — all internet traffic must exit through WireGuard
  • Docker internal — container-to-container communication
  • WireGuard handshake — to re-establish if tunnel drops

Everything else silently dropped. IPv6 fully blocked. Auto-restart reconnects within seconds.

Hardlink Architecture

The single-mount design that eliminates duplicate disk usage.

Works because all containers mount DATA_ROOT as /data. Downloads and media on the same filesystem = kernel can hardlink. A 50 GB file uses 50 GB total, not 100 GB.

Media Pipelines

Each media type has a dedicated pipeline.

🎬 Movies — Radarr (48 Custom Formats)

▼

6 release group tiers, IMAX/Special Edition boosts, 14 unwanted-content rejectors. Cutoff: Bluray-1080p, upgrades indefinitely. Naming follows TRaSH Guide. Categories: torrents → radarr/, usenet → movies/. Post-import: Bazarr fetches subtitles, Jellyfin detects new file.

📺 TV Shows — Sonarr (44 Custom Formats)

▼

Streaming services score +75 each (17 services), DV Boost +1000, HDR +500. Season pack logic: >50% missing → prefer full season. Supports anime with absolute numbering. Cutoff: WEB 2160p.

🎵 Music — Lidarr (5 Custom Formats, Davo's Guide)

▼

CD +500, Preferred Groups (DeVOiD, PERFECT, ENRiCH) +500, WEB +200, Lossless +100, Vinyl -10000. Min score 1 — unknown releases skipped. Cutoff: Lossless. Uses MusicBrainz metadata.

📚 Books & Audiobooks — LazyLibrarian

▼

PostProcessor every 10 min sorts by file type: .epub/.pdf → /books (Kavita reads), .mp3/.m4b → /audiobooks (Audiobookshelf reads). Docker mods add Calibre + FFmpeg. GoodReads metadata.

🎮 Games — QuestArr + game-import.sh

▼

QuestArr is search-only (IGDB metadata). game-import.sh runs every 15 min via cron with 9 safety layers: API completion check, settling time, duplicate detection, empty check, lockfile. Torrent games hardlinked, usenet games moved.

💬 Subtitles — Bazarr

▼

Monitors Radarr/Sonarr for new imports. Queries providers, scores candidates against media metadata (hash, title, year, group). Places .srt alongside media files. Upgrade window: 7 days. Auto-sync timing correction for low-score matches.
